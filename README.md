## 安装与使用

1. **下载游戏**：
    - 通过以下链接下载最新版本的游戏：
      [https://github.com/H-Bole/Boom_VR/releases](https://github.com/H-Bole/Boom_VR/releases)

2. **运行游戏**：
    - 解压下载的文件并运行游戏
    - 打开SteamVR选择社区操作方案
    - 推荐使用PICO VR设备游玩
